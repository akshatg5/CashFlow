## Building a basic version of PayTM - CashFlow

## Setup : 

### Backend
1. cd backend/
2. Make a file .env according to the sample .env.sample file and add a MongoDb URI
3. node index.js
4. Once the backend is up and running, test it on Postman to check it.

### Frontend 
1. cd frontend/
2. npm run dev

Technologies Used :
    ExpressJS
    Zod
    MongoDB
    ReactJS
    Tailwind CSS
